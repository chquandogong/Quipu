# v0.2.0 Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add CI, a read-only one-shot Linux collector, and intervention records that connect Quipu's investigation flow to human actions.

**Architecture:** Keep the current FastAPI + SQLite + React architecture. Add `apps/collector` as a small Python package that emits existing ingest payloads, add intervention persistence in the server repository, and expose intervention state through investigation detail and the React UI.

**Tech Stack:** Python 3.12, FastAPI, SQLite, pytest, Vite React, Vitest, GitHub Actions.
